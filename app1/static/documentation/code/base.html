This is the page where all begins. This is not a tutorial but an explanation for what's happening behind the hood.

You could get all the code in the github link here

If you want to create your own bot, you can create your own UI or take my index.html & style.css file to get started

Your UI must meet the following requirements:
  • A button to start recognizing
  • A text view to show the result
  • A text view to show the recognized text

Linking the files

I hope you are ready with your UI, having the above mentioned requirements. Now let's link our crutial files.

1. Here, we use Artyom.js library to ease our voice command recognition

First of all, download the Artyom.js file from here

After download, place the file into your project's directory and add the script file into head section of your index.html file

<script src="yourproject/artyom.js"></script>

2. Well, we need Jquery for little things. But, I always love playing with vanilla javascript

Copy and paste the below code in head section of your index.html file

copy<script  src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
