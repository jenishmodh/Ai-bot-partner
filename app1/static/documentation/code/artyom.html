In this section, we are going to deal with the artyom.js library.

Introduction

Artyom is a javascript library, created by sdkcarlos in which it uses webkitSpeechRecognition and speechSynthesis APIs in the backend to provide a quick resonsive speech recognition.

Creating instance

I hope you already have empty files named speech.js and data.js in your project. If not, create them.

Now open speech.js and create the instance of artyom file into it (copy the below code and paste it)
